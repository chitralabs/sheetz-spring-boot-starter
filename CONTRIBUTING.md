# Contributing to Sheetz Spring Boot Starter

Thank you for your interest in contributing! Please see the [main Sheetz contribution guidelines](https://github.com/chitralabs/sheetz/blob/main/CONTRIBUTING.md) for the full guide.

## Quick Start

```bash
git clone https://github.com/chitralabs/sheetz-spring-boot-starter.git
cd sheetz-spring-boot-starter
mvn clean install
```

## Pull Requests

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Write tests for your changes
4. Ensure all tests pass: `mvn clean test`
5. Submit a pull request against `main`
